# Todo List

A todo list app using React (CRA) and TypeScript.

### Functionality

- [x] List up todos using input field
- [x] Mark done when todos are complated
- [x] Delete todos once clicking the bin icon
- [x] Update todos when clicking the edit icon
- [ ] Drag & Drop tasks
